# Re-Ranking Pipeline Experiment Scripts

This folder contains experiment scripts for running static, dynamic re-rankers for the candidate list generated by neural models.

---

## Requirements

- Please make sure the models in `experiment_scripts` are already generated results.
- Then you can skip training, ranking of the recommender models.
- In this folder, the pipeline experiment allows loading the saved results in the in-processing stage and directly performs re-ranking.

---

## Configuration

- Change the config files in the `configs` folder. Make sure the `model_ranked_items_path` saves the results from `experiment_scripts`.

---

## Execution Flags

Use the following flags in the `.ini` file to control which stages run:

- `model_action`
- `static_reranking_action`
- `dynamic_reranking_action`
